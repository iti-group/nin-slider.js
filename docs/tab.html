tabの使い方

tabを使うにはclassにnin_tabを指定し、中に2つのdivを作ります。
一つ目のdivにはタブのタイトルを、二つ目のdivにはその内容を入れます。 一つ目のdivに"active"から始まるclassと"inactive"から始まるclassを設定します。
active から始まるclassはそのタブが選択されているの時に有効になりタブのタイトルに適用されます
inactiveから始まるclassはそのタブが選択されていないときそのタイトルに適用されます。

 <div class="nin_tab">
    <div class="active-title inactive-title">
        <div>title1</div>
        <div>title2</div>
        <div>title3</div>
        <div>title4</div>
        <div>title5</div>
    </div>
    <div>
        <div>content1</div>
        <div>content2</div>
        <div>content3</div>
        <div>content4</div>
        <div>content5</div>
    </div>
</div>

デモのコード

 <style type="text/css">
 .nin_tab_titles {
     background-color: blue;
 }
 .active-title, .inactive-title {
     height: 2rem;
     line-height: 2rem;
 }
 .active-title {
     background-color: black;
     color: white;
     border-radius: 1rem;
 }
 .inactive-title:hover {
     border-radius: 1rem;
     background-color: rgba(255,255,255,.5);
 }
 .inactive-title {
     color: #f00;
 }
 </style>
 <div class="nin_tab">
     <div class="active-title inactive-title">
         <div>title1</div>
         <div>title2</div>
         <div>title3</div>
         <div>title4</div>
         <div>title5</div>
     </div>
     <div>
         <div style="background-color: red;">content1</div>
         <div style="background-color: blue;">content2</div>
         <div style="background-color: green;">content3</div>
         <div style="background-color: orange;">content4</div>
         <div style="background-color: gray;">content5</div>
     </div>
 </div>
