Sliderの使い方

Slider

sliderを使うにはclassに"nin_slider"を指定します また"data-option"にJSON形式でオプションを指定できます。

<div class="nin_slider" data-option='{"spd": 5,"auto": true, "auto_interval": 4000}' style="height: 20rem;">
  <div style="background-color: red;"></div>
  <div style="background-color: blue;"></div>
  <div style="background-color: green;"></div>
</div>

spd: スライド速度
auto: 自動でスライドするか
auto_interval: 自動でスライドする場合の間隔(ms)
以下デフォルト設定

{
  "spd": 5,
  "auto": true,
  "auto_interval": 4000
}

スライドアイテムにクリックイベントを設定する場合は"onclick"に設定してください。

  <div id="test" onclick="event"></div>

または

  var element = document.getElementById("test");
  element.setAttribute("onclick", test);

デモのコード 

<div class="nin_slider" data-option='{"spd": 5,"auto": true, "auto_interval": 4000}' style="height: 20rem;">
    <div onclick="alert('a')" style="background-color: red;"></div>
    <div style="background-color: blue;"></div>
    <div style="background-color: green;"></div>
</div>
