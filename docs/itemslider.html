ItemSliderの使い方

ItemSlider

item-sliderを使うにはclassに"nin_item-slider"を指定します また"data-option"にJSON形式でオプションを指定できます。

<div class="nin_item-slider" data-option='{"size": 3,"scrollVolume": 3,"spd":5}' style="height: 200px;">
    <div style="background-color: #800"></div>
    <div style="background-color: #c00"></div>
    <div style="background-color: #f00"></div>
    <div style="background-color: #f40"></div>
</div>

size: 一度に表示されるアイテムの数
scrollVolume: 左右のボタンを押した際動く量
spd: スクロール速度
以下デフォルト設定

{
    "volume": 3,
    "scrollVolume": 3,
    "spd":5
}

jsで要素を追加する場合

このjsの実行準備が完了したときにファンクションを実行するためには以下の方法をとる

nin.onload.push(ファンクション名);

nin.onload 以降に"nin.getItemSliderById(id名)"で"ItemSlider"を取得し、add(element)メソッドで 追加できます。
また、deleteAll()メソッドにて中身のelementをすべて削除できます。

if(nin.onload) nin.onload.push(test1);
else test1();

function test1() {
    var is = nin.getItemSliderById("item-sliderにつけたid名");
    is.deleteAll();//中身削除
    var el = document.getElementById("id");
    is.add(el);//elをitem-sliderに追加
}
