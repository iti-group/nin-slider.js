modalの使い方

modal

modalを使うにはclassに"nin_modal"を指定し、任意のid設定します。 

<div class="nin_modal" id="modal1">modal</div>

表示するにはnin.modal.show(id名)メソッドを使用します。

nin.modal.show("modal1");

閉じる際はnin.modal.close()メソッドを使用します。

nin.modal.close();

デモのコード 

<style type="text/css">
#modal1 {
    background-color: blue;
    width: 50%;
    height: 500px;
}
</style>
<button onclick="nin.modal.show('content1')">クリックで開く</button>
<div class="nin_modal" id="modal1">
    <button onclick="nin.modal.close()">close</button>
</div>
